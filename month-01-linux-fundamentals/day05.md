# Linux Day 05

## Objective

Learned about Linux users, groups, and sudo.

## Commands Learned

- whoami
- id
- groups
- sudo

## Concepts

- User
- Root
- Group
- Privilege Escalation

## Practice

- Identified current user.
- Viewed user and group information.
- Used sudo to execute commands as root.
- Learned the difference between a normal user and root.

## Notes

Root is the most privileged user in Linux. The sudo command allows a normal user to execute commands with elevated privileges.
