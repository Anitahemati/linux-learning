# Linux Day 02

## Commands Learned

### cp
Used to copy files and directories.

Example:
cp file1.txt backup/

### mv
Used to move files/directories or rename them.

Examples:
mv file.txt backup/
mv file.txt newfile.txt

### rm
Used to remove files.

Example:
rm file.txt

### rmdir
Used to remove empty directories.

Example:
rmdir testdir

## Practice

Created directories and files.
Copied files using cp.
Moved and renamed files using mv.
Deleted files with rm.
Removed empty directories with rmdir.

## Notes

- mv can be used for both moving and renaming.
- rmdir only works on empty directories.
- rm removes files permanently.
